## 📌 Project Overview
A comprehensive personal financial budget analysis project built 
entirely in Microsoft Excel. This project analyzes 806 personal 
financial transactions across 19 spending categories over 21 months 
from January 2018 to September 2019, comparing actual spending against 
planned budgets to uncover key financial insights and recommendations.

## 🔍 Analysis Performed

- ✅ Data cleaning and preparation in Excel
- ✅ Helper columns using IF, TEXT, YEAR functions
- ✅ Budget vs Actual comparison using SUMIF and VLOOKUP
- ✅ Variance and Variance % calculation per category
- ✅ Conditional formatting for instant visual insights
- ✅ Pivot Tables for category and monthly analysis
- ✅ Interactive Dashboard with Slicers
- ✅ Business Recommendations Report
